# careers-connect-website

Career Connect is a careers website built with Python, Flask, HTML, Bootstrap, Jinja templates, and SQLite.

## Features
- Displays job openings from a database
- Shows dynamic job detail pages
- Lets users submit applications online
- Stores applications in SQLite

## Future Work

- Add login and admin access
- Allow editing and deleting applications
- Move from SQLite to a production database for deployment
